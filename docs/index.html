SoC Labs Design Contest 2024/25

We are competing in the Hardware Implementation track, designing a mixed-signal SoC that demonstrates how compute makes use of real-world sensor data for precision agriculture. Our design integrates analogue front-end components with a digital processing subsystem on the SoC Labs NanoSoC reusable MCU platform.

System Architecture

Built on the ARM CMSDK and NanoSoC platform, our SoC integrates a Cortex-M0 processor with custom sensor peripherals via an AHB bus interconnect.

System architecture showing the ARM Cortex-M0 core, AHB bus, and peripherals

Digital Subsystem

The digital subsystem is centred on the ARM Cortex-M0 processor, connected via an AHB-Lite bus matrix to custom peripherals including our ADC APB wrapper, FIFO controller, real-time clock, and GPIO. Firmware running on the processor orchestrates sensor data acquisition, processing, and decision-making.

Custom Peripheral: ADC APB Wrapper

A key custom IP block providing the interface between the processor and the analogue front-end. It includes:

  • APB register interface for processor control
  • FIFO buffering for ADC sample data
  • FSM-based wrapper control for sequencing conversions
  • Analogue multiplexer selection for multi-channel sensing
  • Real-time clock for timed/periodic sampling

Analogue Front-End

The analogue subsystem conditions real-world sensor signals for digitisation. It includes filtering, amplification, and sample-and-hold stages modelled in Verilog-AMS for co-simulation with the digital design.

Key Features

Designed from the ground up for reliable, energy-efficient field deployment.

Multi-Sensor Integration

Supports electrochemical, temperature, and dielectric sensors through a flexible analogue multiplexer and ADC pipeline, enabling comprehensive soil health monitoring from a single device.

Mesh Network Architecture

Devices form a self-healing mesh network, eliminating single points of failure and enabling coverage across large agricultural areas without centralised infrastructure.

Energy Efficient Design

Engineered for ultra-low power operation to sustain continuous monitoring throughout an entire crop season on limited energy budgets, using timed wake-up and duty-cycled sampling.

Real-Time Data Acquisition

On-chip FIFO buffering and RTC-driven sampling provide continuous, granular data directly from the field, surpassing traditional remote sensing approaches like drone mapping.

NanoSoC Platform

Built on the SoC Labs NanoSoC reusable MCU platform with ARM Cortex-M0, DMA controllers, and a proven debug interface, reducing development risk and accelerating verification.

Mixed-Signal Co-Design

Analogue front-end modelled in Verilog-AMS alongside digital RTL, enabling full-system behavioural simulation with realistic sensor signal characteristics.

Project Milestones

Tracking our progress from initial research through to silicon-ready design.

June 2024

Getting Started Completed

Determined project scope, voted on precision agriculture application, completed ARM introductory SoC course, reviewed NanoSoC reference design.

July 2024

IP Selection Completed

Identified required processor and peripheral features. Created draft architectural diagram and determined sensor requirements for soil monitoring.

August 2024

SoC Specification Completed

Defined internal module layout, communication specifications (APB, AHB), and adopted ARM DesignStart IP as bare-bones starting point.

June 2025

RTL Generation Completed

Created APB wrapper for custom peripherals, migrated to NanoSoC with AAA IP, defined memory map and address allocation. Achieved working NanoSoC + custom peripheral simulation on Icarus Verilog.

July 2025

Simulation Completed

Completed integration test simulation of custom RTL with NanoSoC.

September 2025

Cadence Workflow Familiarisation Completed

Gained proficiency with full Cadence Design Tools workflow for synthesis and implementation. Documented step-by-step guide for the team.

January 2026

Technology Selection Completed

Selected SKY130 open-source PDK with Cadence MPW flow after evaluating TSMC 65nm option. Ready for physical implementation.

February 2026

Behavioural Modelling In Progress

Developing complete Verilog-AMS behavioural model of the analog subsystem including filter, amplifier, and sample-and-hold modules for integration with NanoSoC.

February 2026

RTL Verification In Progress

Verifying custom RTL + NanoSoC on FPGA with mock ADC input. 5 firmware tests simulated; investigating RTC bug.

March 2026

Design for Test Upcoming

Adding debug counters and readout support to custom RTL for post-silicon diagnostics.

March 2026

Synthesis Upcoming

Synthesising the full digital design (custom RTL + NanoSoC) using Cadence Genus with the SKY130 PDK.
